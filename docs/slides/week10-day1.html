Two-Way ANOVA Models

Two-way ANOVA


Goal:

Assess if multiple categorical variables have a relationship with the response.

Modeling Options

Additive Model

Assess if each explanatory variable has a meaningful relationship with the response, conditional on the variable(s) included in the model.

Interaction Model

Assess if the relationship between one categorical explanatory variable and the response differs based on the values of another categorical variable.

What are we looking for?

Hypothesis Test Steps

Step 1: Fit a one-way ANOVA model for each categorical variable

Step 2: Decide if each explanatory variable has a meaningful relationship with the response variable

  • If yes, then go to step 3!
  • If no, then report which variable (if any) has the strongest relationship with the response.

What about two-way ANOVA interaction models?

  • An interaction model is similar to a different slopes multiple linear regression

  • It assesses if the relationship between two variables differs when a third variable is considered

  • Due to time, we we are not fitting interaction models for the Final Project

ANOVA quantifies how much each explanatory variable in a statistical model contributes toward explaining the total variation in the response variable

Total Variation in a One-way ANOVA


age_lm <- lm(score ~ age_cat, data = evals_small) 

age_anova <- anova(age_lm) %>% 
  tidy() 

age_anova_total <- tibble(term = "Total", 
                          df = nrow(evals_small) - 1, 
                          sumsq = sum(age_anova$sumsq)
                          )
age_anova_full <- age_anova %>% 
  bind_rows(age_anova_total) 

age_anova_full %>% 
  gt::gt()
term df sumsq meansq statistic p.value
age_cat 3 0.8903343 0.2967781 0.9454526 0.4221873
Residuals 90 28.2510487 0.3139005 NA NA
Total 93 29.1413830 NA NA NA


. . .

The age_cat line in the ANOVA table quantifies how much of the total variation in minimum evaluation scores can be explained based on the age group of a faculty member.

The Total row in an ANOVA table tells us the total variation in evaluation scores – how far each faculty member’s minimum evaluation score was from the mean of all the evaluation scores

Connection to \(R^2\)

Based on the ANOVA table, there is 29.1414 total variation in minimum evaluation scores, of which age_cat is able to explain 0.8903.

. . .

Proportionally, that means age_cat can explain 3.055% in the total variation in minimum evaluation scores.
